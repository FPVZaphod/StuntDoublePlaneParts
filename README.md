# StuntDoublePlaneParts
Parts for 3d printed planes of Stunt Double

CanopyO4ServoFront.stl : A front canopy part for a 9g pan servo and a DJI O4. This file is for one VTX Antenna. 
DualAntenna-V-Adapter_v2.stl:  A small Adapter to use two O4-Antennas on CanopyO4ServoFront.stl
O4-ServoPan_v2: To mount a DJI O4 Camera on a 9g Pan Servo
